# Illyria High: Read the Room

A visual novel choice-and-consequence game designed for Year 11 English students studying *She's the Man* and social texts.

## Architecture
- Vanilla HTML/CSS/JS. No frameworks.
- Data-driven structure (`data/` folder) allowing easy future adaptation for *The Taming of the Shrew*.
- Base resolution: 1280x720, responsive container scaling.
- Live DOM rendering without canvas.

## Running the game
Simply open `index.html` in a modern web browser. Double-click the file or use a local development server.
